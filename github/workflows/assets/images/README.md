# AdReward IR 🚀

## ویژگی‌ها
- 💰 سیستم پاداش پلکانی (نفر اول بیشترین)
- 🏆 Leaderboard رقابتی
- 📊 داشبورد شفافیت
- ⏰ هشدار Screen Time
- 👥 سیستم ریفرال
